DwellSense

Don't sign a blind lease.

DwellSense is a full-stack forensics platform that scans any NYC address and exposes crime, complaints, permits, evictions, transit gaps, and flight noise — so renters know the truth before they sign.

Built with Next.js, FastAPI, Mapbox GL, and Google Gemini AI. Pulls live data from NYC Open Data, Google Places, and FAA flight corridors. Deployed on Vercel + Railway.

Features
🔍

Address Scanner

One address. Full forensics.

Enter any NYC property address and DwellSense fires parallel API calls to pull crime reports, 311 complaints, building permits, eviction records, transit/grocery proximity, and flight path data — returning a complete threat profile in seconds.

🚨

Results Dashboard

Danger score, map, and logistics in one view.

The dashboard opens with a color-coded risk banner (LOW / MODERATE / HIGH / EXTREME) and a composite 0–100 danger score. Below it, a horizontal carousel shows nearest subway, bus, airport, and grocery with exact distances. The interactive Mapbox map plots every incident as a hoverable emoji pin with color-coded threat zones.

📑

9-Point Threat Analysis

AI-written intel on every angle.

Nine swipeable cards cover tenant churn, police calls, area safety, tenant warnings, demolitions, noise schedule, flight paths, 311 reports, and the oven effect. Each card gets three data-driven bullets — written by Google Gemini 2.0 Flash using the address's real data, not templates.

📄

PDF Dossier & Full Map

Download the complete report.

One click generates a full PDF with danger score, all nine threat cards, and logistics data. The interactive map supports scroll-to-zoom, hoverable incident pins with popups, animated flight path with a moving plane icon, and route lines that draw when you hover a logistics card.

Tech Stack & APIs
⚛️
Next.js
Frontend Framework
React-based App Router with server components, API route proxying, and Tailwind CSS for the entire UI.
⚡
FastAPI
Backend API
Python async backend running all data lookups in parallel with asyncio.gather, serving the /scan endpoint.
🌎
Mapbox GL JS
Interactive Map
Dark-themed vector map with custom emoji markers, GeoJSON zones, animated flight paths, and route overlays.
🤖
Google Gemini 2.0
AI Analysis
Writes 27 threat-card bullets from real crime/311/permit data. Falls back to data-driven templates when unavailable.
📊
NYC Open Data
SODA API
Live crime reports, 311 service requests, building permits, and eviction filings queried via Socrata endpoints.
📍
Google Places
Logistics Data
Nearest subway, bus, airport, and grocery locations with distance calculations for the logistics carousel.
✈️
FAA Corridors
Flight Analysis
Computes proximity to NYC flight approach/departure corridors and renders animated plane paths on the map.
🚀
Vercel + Railway
Deployment
Frontend on Vercel with edge routing. Python backend on Railway with environment-managed API keys.
